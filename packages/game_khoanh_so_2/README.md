# Game Khoanh Số 2

## Mô tả

Game yêu cầu người chơi tìm và khoanh tròn vào **cả 2 đáp án đúng**.

- Mỗi lần khoanh chỉ được chọn 1 đáp án.
- Cho phép đè lên hình khác tối đa 20%.
- Khoanh vùng trống hoặc sai đáp án sẽ bị báo lỗi.
